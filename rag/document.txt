✅ Key Notes (Important to Know)

There is no single static schema file for Optimizely SaaS CMS
Every environment has a dynamically generated schema
Best practice is to:

Use GraphiQL Docs tab
Or run GraphQL introspection queries


Example introspection query:
GraphQL{}  __schema {    types {      name      kind    }  }Show more lines
