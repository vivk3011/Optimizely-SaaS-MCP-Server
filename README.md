# Optimizely SaaS MCP Server (Graph + CMA REST API)

This project provides MCP tools for both **reading** (via Optimizely Graph) and **writing** (via CMA REST API) to Optimizely SaaS CMS.

## How to Run

### Build the repo (Read-Only)
- Clone the Repo e.g. C:\Projects\Optimizely Saas\
- run npm install
- post npm install, make sure to have 2 files 1)cd-graph-index.js 2) cm-restapi-index.js under \dist folder
- copy folder .github and paste it under your Optimizely SaaS Project soluction
